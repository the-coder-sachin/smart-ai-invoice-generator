**Smart AI Invoice Generator** is a full-stack web application designed to help businesses quickly create professional invoices with AI-powered assistance.

Built on the MERN stack (MongoDB, Express, React, Node.js), this project integrates **Google Gen AI** to automate invoice content formatting, generate reminders, text-to-invoice creation, total revenue calculation, and business insights.

## üì• Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/the-coder-sachin/smart-ai-invoice-generator.git
cd smart-ai-invoice-generator

# 2. Install backend dependencies
cd backend
npm install

# 3. Install frontend dependencies
cd ../frontend
npm install

# 4. Configure environment variables
#    - AI API key / credentials for Google Gen AI
#    - MongoDB connection string
#    - JWT secret
```
